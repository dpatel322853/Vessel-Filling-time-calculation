# Ethylene Vessel Filling Toolkit

Updated features:
- Interactive 3D plot in Streamlit app: Time vs Vessel Pressure vs RO Flowrate.
- PDF report export from Streamlit app including inputs, KPIs, 2D plots, 3D plot, and selected result rows.

Run:
```bash
pip install -r requirements.txt
streamlit run ethylene_filling_streamlit_app.py
```
